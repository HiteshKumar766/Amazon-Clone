# Amazon Clone

A simple Amazon-inspired homepage clone built using **HTML** and **CSS**. This project was created for learning and practicing front-end web development concepts such as layouts, styling, responsiveness, and component design.


## 🎯 Learning Objectives

Through this project, I practiced:

- Semantic HTML structure
- CSS Flexbox
- CSS Grid
- Responsive web design
- Layout positioning
- Styling reusable components

## ⚠️ Disclaimer

This project is created for educational and practice purposes only. It is inspired by the Amazon website design and is not affiliated with, endorsed by, or connected to Amazon in any way.
